# Testing Repository for "Move Issue on Assign" Action

This repository is a dedicated testing environment for the **[Move Issue on Assign](https://github.com/immitsu/move-issue-on-assign)** GitHub Action.

## Purpose

- To test the Action's functionality in an isolated environment.
- To reproduce and debug issues.
- To experiment with different configurations and workflows.

## How to Use

1. Create test issues in this repo.
2. Assign them to users.
3. Observe the automatic movement between project columns.
